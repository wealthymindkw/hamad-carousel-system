## ⚡ البداية السريعة

### الطريقة 1: التطبيق المرئي (الأسهل)
```bash
# افتح الملف في المتصفح مباشرة
open app/weekly_carousel.html
```
ثم اضغط **"ولّد كاروسيل هذا الأسبوع"** 🎯

### الطريقة 2: Python Script
```bash
# ثبّت المتطلبات
pip install requests python-dotenv

# أضف مفتاح Anthropic
cp .env.example .env
# عدّل .env وأضف ANTHROPIC_API_KEY

# شغّل السكريبت
python scripts/generate_carousel.py
```

## 📋 هيكل الكاروسيل

كل كاروسيل يتكون من **6 سلايدات** بهذا الترتيب:

| # | النوع | المحتوى |
|---|-------|---------|
| 1 | 🎯 كفر (Hook) | قصة شخصية + كشف الموضوع |
| 2 | 💡 فكرة 1 | مبدأ + جملة قوية بين "" |
| 3 | 💡 فكرة 2 | ثلاثية / إطار عمل |
| 4 | 💡 فكرة 3 | تحدي أو عائق |
| 5 | 💡 فكرة 4 | خطوة عملية + سؤال |
| 6 | 📣 CTA | اكتبلي (تحدي) بالتعليقات |
