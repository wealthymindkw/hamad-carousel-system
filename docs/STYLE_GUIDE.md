## هيكل السلايدات

### سلايد 1 — الكفر (Hook)
```
[سيناريو شخصي محدد بتفاصيل]
[فجوة أو مشكلة واضحة]

[اسم الموضوع بالخط العريض]

[نتيجة مباشرة أو وعد]
```

**مثال:**
```
كنت أشتغل 12 ساعة يومياً…
بس إنجازي ما يتغير.

اكتشفت شي اسمه
قانون التركيز

من يومها إنجازي تضاعف بنصف الوقت.
```

---

### سلايدات 2-5 — المحتوى
```
[عنوان الفكرة]:
"[جملة قوية تلخص الفكرة]"

[شرح مختصر 3-4 أسطر]
[تطبيق عملي أو مثال]
```

**مثال:**
```
التركيز مو مهارة، هو قرار:
"ما تقدر تركز على كل شي…
اللي يركز على كل شي، ما يركز على شي."

اختار 3 أشياء فقط يومياً.
باقي الأشياء: تأجيل بوعي.
```

---

### سلايد 6 — CTA
```
[جملة تعاطف مع القارئ]
[مشكلته بكلمات بسيطة]

[وصف التحدي باختصار]
[قيمة واضحة]

اكتبلي (تحدي) بالتعليقات وراح ارسل لك رابط التحدي
```

## المواضيع الأساسية

| الركيزة | أمثلة المواضيع |
|---------|----------------|
| 🧠 الوعي | الهوية الخفية، الصمت الداخلي، الراصد والمخطط |
| 💰 الوفرة | طاقة المال، قانون الاستحقاق، ذهنية الندرة |
| 🌌 القوانين الكونية | قانون الجذب، الإيقاع، السبب والنتيجة |
| 🔄 تحول الهوية | نسختك الجديدة، بناء الشخصية، كسر القناعات |
| ⚡ الإنتاجية | التركيز العميق، الطاقة لا الوقت، أنظمة العمل |

## الإطار الثلاثي (الأقوى)

استخدم دائماً أسماء لأجزاء الإطار:

```
ثلاثية [اسم الموضوع]:

الراصد: ...
المصفّي: ...
المختار: ...
```

أمثلة أخرى:
- الباني، المحرر، الموسع
- الجذب، الإيقاع، التجلي
- الوعي، القرار، الفعل
